
# Customer Update

Represent an update to customer fields

## Structure

`CustomerUpdate`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `firstName` | `string \| undefined` | Optional | First name(s) / given name(s) |
| `lastName` | `string \| undefined` | Optional | Last name(s) / surname(s) |

## Example (as JSON)

```json
{
  "firstName": "John",
  "lastName": "Smith"
}
```
