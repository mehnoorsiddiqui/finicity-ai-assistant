
# Birthday

A birth date

## Structure

`Birthday`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `year` | `number \| undefined` | Optional | The birthday 4-digit year |
| `month` | `number \| undefined` | Optional | The birthday 2-digit month (1 is January) |
| `dayOfMonth` | `number \| undefined` | Optional | The birthday 2-digit day-of-month |

## Example (as JSON)

```json
{
  "year": 1989,
  "month": 8,
  "dayOfMonth": 13
}
```
